# План визуализации данных (Инфографика)

## 1. Главный дашборд (Overview)
| Виджет | Тип графика | Данные | Цель |
| :--- | :--- | :--- | :--- |
| **Общий охват** | Line Chart | Просмотры по дням | Динамика популярности |
| **Engagement Rate** | Gauge Chart | Средний % вовлеченности | Качество контента |
| **Баланс категорий** | Donut Chart | Кол-во постов по категориям | Контроль контент-плана |

## 2. Аналитика постов
*   **Сравнение площадок (Bar Chart):** Столбчатая диаграмма, где для каждого поста сравниваются лайки в VK и Telegram.
*   **Облако тегов/категорий:** Визуализация самых часто используемых тем, где размер зависит от среднего ER.

## 3. Календарь (Content Calendar)
*   **Интерактивная сетка:** Возможность Drag-and-drop для переноса даты публикации.
*   **Цветовое кодирование:**
    *   🔵 #3b82f6 — Мероприятия
    *   🟢 #10b981 — Гранты
    *   🟡 #f59e0b — Новости
    *   🔴 #ef4444 — Срочное / Важное

## 4. Отчеты для руководства
*   **Экспорт в PDF:** Генерация страницы с ключевыми графиками и Топ-3 постами месяца.
*   **Сводная таблица:** Сравнение текущего месяца с предыдущим (прирост в %).

## 5. UI Компоненты (Frontend)
*   **Библиотека:** Recharts (для React).
*   **Стиль:** Glassmorphism или чистый Flat Design (shadcn/ui).
*   **Тема:** Поддержка темной и светлой темы для комфортной работы в любое время.
